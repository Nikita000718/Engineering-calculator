# 🧮 Инженерный калькулятор на Python (Tkinter)

Профессионально оформленный графический инженерный калькулятор с поддержкой математических функций, истории вычислений, динамического превью и адаптивной темной/светлой темой.

Специально оптимизирован для корректного отображения цветов интерфейса на **macOS** с помощью библиотеки `tkmacosx`.

---

## ✨ Основные возможности

* **Инженерные функции:** Вычисление тригонометрии (`sin`, `cos`, `tan`), логарифмов (`log`, `ln`), факториала (`fact`), извлечение квадратного корня (`√`), возведение в степень (`ˆ`) и работа с числом Пи (`π`).
* **Динамическое превью (Live Preview):** Результат вычисляется на лету и отображается серым шрифтом над основным дисплеем прямо во время ввода выражения.
* **Умный парсер скобок:** Автоматически закрывает недостающие скобки при вычислении или просмотре превью.
* **Текстовая история вычислений:** * Все успешные вычисления сохраняются локально в файл `history.txt`.
    * Кнопка **H (History)** открывает отдельное окно со списком прошлых операций.
    * **Двойной клик** по строке истории автоматически возвращает выражение на главный дисплей для продолжения работы.
* **Полноценная память:** Кнопки `MC` (очистить), `MR` (вызвать), `M+` / `M-` (добавить/вычесть).
* **Смена тем оформления:** Кнопка со смайликом (`🌙` / `☀️`) мгновенно переключает калькулятор между Dark и Light темами с полной перекраской всех элементов интерфейса.

---

## 🛠️ Стек технологий

* **Язык:** Python 3
* **Графический интерфейс:** `tkinter`
* **Оптимизация для macOS:** `tkmacosx` (динамический фоллбэк на стандартный `tkinter` для Windows/Linux пользователей).
* **Математический движок:** встроенный модуль `math`.
